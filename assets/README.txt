Place shared assets here:
  - favicon.ico / favicon.png
  - og-image.jpg  (Open Graph preview image, 1200x630 recommended)
  - Any SVG icons or custom fonts

To add a favicon, drop favicon.ico here and add to <head> in index.html:
  <link rel="icon" href="assets/favicon.ico" />

To add an OG image for social sharing:
  <meta property="og:image" content="assets/og-image.jpg" />
